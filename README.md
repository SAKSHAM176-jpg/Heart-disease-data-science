# Heart-disease-data-science
Predicting heart disease using machine learning with Python libraries like NumPy, Pandas, Matplotlib, and Scikit-learn. Based on UCI Cleveland Heart Disease dataset.

This project explores how machine learning and data science techniques can be applied to predict the likelihood of heart disease in patients based on clinical parameters.
Using Python libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn, the notebook demonstrates the complete workflow of building, evaluating, and experimenting with classification models.
